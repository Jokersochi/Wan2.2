## 2025-03-09 - Adding descriptions to CLI progress bars
**Learning:** For CLI-based Python tools, especially those involving long-running processes like video generation or preprocessing steps, adding descriptive text (e.g., `desc="..."`) to `tqdm` progress bars significantly improves the user experience. It provides clear, contextual feedback about what the system is currently doing, making the wait time feel more manageable and informative.
**Action:** Always include a relevant description when implementing or updating `tqdm` progress bars in CLI applications.
